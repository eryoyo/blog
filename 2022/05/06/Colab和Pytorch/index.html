Colab和Pytorch

李宏毅课程中Colab, Pytorch总结

Colab

  1. 点击+ Code来添加code cell，点击+ Text来添加text cell，还有将cell上移以及下移的选项，可以将cell删除
  2. 在code cell中有两种命令可以输入，一种是python代码，直接输入即可；另一种是shell命令，在命令的前面需要加上!
  3. 命令的前面加上!，会先执行这个命令，之后就会杀掉这个进程，而在命令的前面加上%就会影响和这个jupyter相关的进程，称为magic command，可以查看https://ipython.readthedocs.io/en/stable/interactive/magics.html来知道更多的magic command
  4. 更换为使用GPU：点击Runtime-点击Change Runtime Type-选择GPU，更换runtime的操作会重启这个session
  5. 点击cell前面的运行按钮就会执行这个cell的代码，也可以点击runtime里面的选项来运行代码
  6. 查看抽到的GPU类型!nvidia-smi，有三种可能性P100>T4>k80
  7. 通过google drive下载文件，google drive分享的文件如下链接https://drive.google.com/open?id=xxxxxx，这个xxxxxx就是文件ID，使用下列命令来下载`!gdown –id ‘xxxxxx’ –output pikachu.png`
  8. 下载之后的文件可以通过点击文件夹图标查看，这些文件都是暂时储存的，在session结束之后就会被清理
  9. 可以点击上载图标来上传本地文件，可以下载文件
  10. 可以不用每次都上传以及下载文件，直接将google drive挂载到colab即可，首先导入包from google.colab import drive之后挂载drive.mount('/content/drive')
